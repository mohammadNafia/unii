# Deploy to Streamlit Cloud

## Quick Deploy

1. **Go to [Streamlit Cloud](https://streamlit.io/cloud)**
2. **Sign in with GitHub**
3. **Click "New app"**
4. **Select your repository:** `mohammadNafia/unii`
5. **Main file path:** `diabetes_prediction_app.py`
6. **Python version:** 3.8 or higher
7. **Click "Deploy"**

## Requirements

Your repository already has:
- ✅ `diabetes_prediction_app.py` (main app file)
- ✅ `requirements.txt` (dependencies)
- ✅ `model.pkl` and `scaler.pkl` (model files)

## Alternative: Deploy via GitHub

1. Fork or use your existing repository
2. Go to https://share.streamlit.io/
3. Connect your GitHub account
4. Select `mohammadNafia/unii`
5. Set main file: `diabetes_prediction_app.py`
6. Deploy!

Your app will be live at: `https://[your-app-name].streamlit.app`
